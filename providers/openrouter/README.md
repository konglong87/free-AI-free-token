## 一句话说明

OpenRouter 提供 OpenAI-compatible API，可通过统一接入地址调用多个 `:free` 免费模型。

## 快速开始

1. 打开 OpenRouter 官网并注册账号。
2. 在控制台创建 API key。
3. 选择模型名带 `:free` 的免费模型。
4. 在客户端中配置接入地址、API key、模型名。
5. 按需查看 `integrations/` 下的客户端接入文档。

## 核心信息

- 官网：https://openrouter.ai/
- 控制台：https://openrouter.ai/settings/keys
- 文档：https://openrouter.ai/docs/quickstart
- 接入地址：https://openrouter.ai/api/v1
- 推荐免费模型：优先从官方模型列表筛选 `:free`
- 是否 OpenAI-compatible：是
