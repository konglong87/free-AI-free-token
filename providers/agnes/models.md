## 能力范围

截图中明确提到 Agnes AI API 可用于：

- AI 对话与文本生成
- 逻辑推理与内容理解
- 文生图与图像编辑
- 图生视频与视频生成
- 音视频同步生成
- Agent 工具与自动化工作流
- 创意内容生成与多模态交互应用
